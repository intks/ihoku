# useUpdateEffect

A hook that works like `useEffect`, but skips the first render. Useful when you want to run an effect only when dependencies change, not on the initial mount.

## Import

```ts
import { useUpdateEffect } from '@intks/ihoku'
```

## API

```ts
useUpdateEffect(effect: React.EffectCallback, deps?: React.DependencyList): void
```

**Parameters**

- **effect**: `React.EffectCallback` - Required. The effect function to run.
- **deps**: `React.DependencyList` - Optional. Array of dependencies.

**Returns**

- Returns `void`.

## Example

```tsx
import { useState } from 'react'
import { useUpdateEffect } from '@intks/ihoku'

const Example = () => {
  const [count, setCount] = useState(0)

  useUpdateEffect(() => {
    console.log('Count changed (not on mount):', count)
  }, [count])

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  )
}
```
