# useOnline

A hook to track the online/offline status of the browser. Automatically updates when the network status changes.

## Import

```ts
import { useOnline } from '@intks/ihoku'
```

## API

```ts
useOnline(): boolean
```

**Returns**

- Returns `boolean` - `true` if the browser is online, `false` if offline.
