# useMount

A hook that runs a callback function only once when the component mounts. Equivalent to `useEffect` with an empty dependency array.

## Import

```ts
import { useMount } from '@intks/ihoku'
```

## API

```ts
useMount(callback: () => void): void
```

**Parameters**

- **callback**: `() => void` - Required. The function to run on mount.

**Returns**

- Returns `void`.

## Example

```tsx
import { useMount } from '@intks/ihoku'

const Example = () => {
  useMount(() => {
    console.log('Component mounted')
    // Perform initialization tasks
  })

  return <div>Component content</div>
}
```
