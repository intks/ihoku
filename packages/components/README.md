# @intks/ihoku

A collection of useful React components built with React, TypeScript and MUI.

## Installation

```bash
npm install @intks/ihoku-components
# or
yarn add @intks/ihoku-components
# or
pnpm add @intks/ihoku-components
```

## Documentation

For detailed API documentation and examples, please visit: [ihoku Documentation](https://intks.github.io/ihoku/)

## Peer Dependencies

- `react >= 18`
