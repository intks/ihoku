# useWindowFocus

A hook to track whether the browser window is focused. Useful for pausing/resuming animations, stopping timers, or showing notifications when the user returns.

## Import

```ts
import { useWindowFocus } from '@intks/ihoku'
```

## API

```ts
useWindowFocus(): boolean
```

**Returns**

- Returns `boolean` - `true` if the window is focused, `false` if blurred.

## Example

