# useOrientation

A hook to track the device screen orientation. Returns the current angle and orientation type.

## Import

```ts
import { useOrientation } from '@intks/ihoku'
```

## API

```ts
useOrientation(): OrientationState | null
```

**Returns**

- Returns an `OrientationState` object or `null` if not supported:
  - **angle**: `number` - The orientation angle in degrees (0, 90, 180, 270).
  - **type**: `OrientationType` - The orientation type (`'portrait-primary'`, `'portrait-secondary'`, `'landscape-primary'`, `'landscape-secondary'`).
